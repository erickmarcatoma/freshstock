# FreshStock 📦⚡

FreshStock is a simple, smart inventory dashboard that replaces messy spreadsheets with an automated priority alert system. It instantly highlights what needs your attention today so nothing spoils or runs out.

## ✨ Features
* 🔴 **Critical Alerts:** Automatically flags items that are expired or low on stock *and* expiring soon.
* 🟡 **Smart Categorization:** Groups items by risk level so you know what to resolve first.
* 🧹 **Clean View:** Hides healthy stock to keep your dashboard clutter-free.
* ⚡ **Zero Setup:** Built with pure HTML, CSS, and vanilla JavaScript—no installation required.

## 🚀 How to Run
1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Edit the `rawInventory` array in `script.js` to add your own items!
